# v0.1 — Foundation

First public release of HomeLab: a reproducible, single-host **NixOS** homelab operated through
**Git**, a Go **control-api**, and a **React** web UI.

## Highlights

- **Declarative NixOS host** via a single flake output `nixosConfigurations.homelab`, parameterized
  entirely by `.env` (read at evaluation time via `HOMELAB_ENV`).
- **Composable modules**: `networking`, `ssh`, `docker`, `tailscale`, `apps`, `control-api`,
  `auth`, `backup`, `platform`, `secrets`.
- **Declarative apps**: drop an `apps/*.nix` file and `modules/apps.nix` turns it into a systemd
  service. Runners: `process`, `compose`, `dockerfile`, `image`.
- **Platform V2 manifests**: `config/{platform,policies,catalogs}.nix` validated at build and
  published read-only to `/etc/homelab/*.json`.
- **Operational control plane**: `control-api` (Go, no external deps) + a React 18 / Vite / TS
  single-page UI for status, actions, deployments, app lifecycle, storage, policy, backups, audit.
- **Authenticated by design**: `oauth2-proxy` (GitHub org/user allowlist, required) in front;
  `control-api` binds loopback; exposed over the tailnet via `tailscale serve` (HTTPS).
- **GitOps deploy with rollback guard**: push `main` → GitHub Actions over Tailscale →
  `nixos-rebuild switch` with armed auto-rollback on failed health check.
- **Auditable**: every mutation journaled to `/var/lib/homelab/*.jsonl`.

## Documentation

- Rewritten English README with banner, badges, and an architecture diagram.
- Full `docs/` set in English with Mermaid diagrams (architecture, deploy flow, API sequence,
  app-lifecycle state) and a browsable GitHub Wiki mirror.

## Notes

- Secrets ship as examples only (`secrets/homelab.example.yaml`, placeholder `.sops.yaml`).
  Generate your own age key and encrypt real values with SOPS.
- Observability tooling (Grafana/Prometheus/…) is intentionally **not** bundled — declare it as an
  app if you want it.
